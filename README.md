README – TechConnect Academy Signup Page
Project Overview

This project was created as part of the practical task: “Assemble a multimedia/web-based application including scripts.”

The objective was to design and develop a basic web-based application using scripting languages such as HTML, CSS, and JavaScript. The application needed to include a signup form with validation and a visually appealing user interface.

My solution is called TechConnect Academy, a responsive signup page for an online coding and digital skills learning platform.

Purpose of the Project

The purpose of this project was to demonstrate the ability to:

Create a multimedia/web-based application
Use scripting languages in a practical environment
Design a responsive and visually appealing user interface
Implement form validation using JavaScript
Combine structure, styling, and functionality into one working website
Technologies Used
HTML

HTML was used to create the structure of the webpage, including:

Form fields
Buttons
Headings
Sections and layout containers
CSS

CSS was used to style the application and improve the visual appearance by adding:

Custom colors and gradients
Responsive layout design
Shadows and transitions
Modern card styling
Mobile-friendly responsiveness
JavaScript

JavaScript was used to add interactivity and scripting functionality, including:

Form validation
Password validation rules
Success message handling
Show/hide password feature
Features Implemented
Signup Form Inputs

The signup form includes the following required fields:

Name
Surname
Email Address
South African ID Number
Password
Validation Features

The application validates user input to ensure correct data entry.

Password Requirements

The password must:

Be at least 8 characters long
Include at least 1 uppercase letter
Include at least 1 number
ID Validation

The South African ID field only accepts:

Exactly 13 digits
Email Validation

The email field checks for a valid email format.

User Interface Design

The website was designed to be modern and visually appealing.

Design elements include:

Dark gradient background
Purple and teal accent colors
Split-screen desktop layout
Responsive mobile layout
Smooth hover and transition effects
Styled form cards with shadows
Additional Features
Icons inside input fields
Show Password toggle feature
Success confirmation message after registration
“Register Another Account” button
Responsive design for desktop and mobile devices
SEO metadata for title and description
Conclusion

This project successfully demonstrates the creation of a multimedia/web-based application using HTML, CSS, and JavaScript. It combines front-end design, scripting, and responsive development to produce a functional and visually appealing signup page.
